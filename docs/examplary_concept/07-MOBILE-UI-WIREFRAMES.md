# Mobile UI Wireframes: Conversational Expert Experience

## Design Philosophy

The UI should feel like **texting a knowledgeable friend** who happens to be a paint expert.

- Not a form
- Not a chatbot
- Not a wizard
- A fluid conversation that adapts to what you share

## Visual Design Notes

### Color Scheme (From Brand)
- Primary: Teal (#0D9488 range)
- Secondary: Purple (#7C3AED range)
- Background: Dark (#0A0A0A)
- Cards: Glass effect with subtle gradients
- Accents: White text, subtle borders

### Typography
- Headers: Bold, large
- Body: Regular, readable
- Labels: Small caps or muted

### Interaction Patterns
- Tap targets: Minimum 44px
- Cards: Rounded corners, subtle shadow
- Animations: Smooth, not distracting
- Loading states: Skeleton or shimmer

### Trust Elements
- вң“ icon for confirmed info
- ~ icon for inferred info
- ? icon for uncertain info
- Confidence bars where appropriate
